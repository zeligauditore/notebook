# Generar una nueva clave SSH

ssh-keygen -t rsa -b 4096 -C "your_email@example.com"
1

Cuando se te indique "Ingresar un archivo donde guardar la clave", presiona Intro. Al hacerlo aceptas la ubicación predeterminada del archivo.

eval "$(ssh-agent -s)"

ssh-add ~/.ssh/id_rsa
1
2
3

Copiar el contenido del archivo id_rsa.pub dentro de la carpeta home/USER/.ssh/ e ingresarlo en BitBucket (opens new window)
